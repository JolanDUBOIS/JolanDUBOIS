## 🛠️ Selected Projects

- **[Sport Index (WIP)](https://github.com/JolanDUBOIS/sport-index)**
  A unified Python SDK providing a coherent, object-oriented API for exploring multi-domain sports data. Features a graph-like navigation model, lazy-loading for network efficiency, and a built-in VCR-style system for deterministic testing and mocking. *(Personal project, package PyPI release coming soon)*

- **[Sports Calendar (WIP)](https://github.com/JolanDUBOIS/football-calendar)**
  Command-line tool that creates personalized sports calendars and syncs them with Google Calendar. Built on top of the **sport-index** SDK, it automates event tracking across all supported sports through a single, streamlined data interface. *(Personal project)*

- **[ECMWF Utils](https://github.com/JolanDUBOIS/ecmwf-utils)**
  A Python utility suite for automating the retrieval and preprocessing of meteorological data from ECMWF services. Developed as part of my **Master's Thesis at NTNU** to streamline atmospheric data integration for ML pipelines. *(Academic project)*

- **[Interactive Quiz System](https://github.com/Harry-Potter-Quiz-Project/harry-potter-quiz)** 
  Full-stack web prototype that manages quiz logic, scoring, and state. Demonstrates client–server architecture, frontend–backend interaction, and modular feature implementation. *(Personal project)*

- **[News Recommendation System](https://github.com/JolanDUBOIS/grp-project-recommender-systems)**  
  Multi-model recommender system built on the MIND dataset. Includes data normalization, evaluation pipelines, and hybrid model implementations. *(Academic project)*
